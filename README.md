# Predictive-Customer-Lifetime-Value-CLV

## Dataset
Online Retail Transaction Data

## Objective
Predict customer lifetime value using RFM features and machine learning.

## Features Used
- Recency
- Frequency
- Monetary Value

## Tools
- Python (pandas, scikit-learn)
- Looker Studio
  Report link: https://lookerstudio.google.com/s/rFomH6XGbGU

## Outcome
Identified high-value customers for targeted retention and marketing strategies.
